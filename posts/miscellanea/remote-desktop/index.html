Remote Desktop

    远程服务器设置

    第一步：在台式机上“允许被连接”

    你需要先告诉台式机：你是可以被远程访问的。

    1. 在台式机上，打开 设置 (Settings) > 系统 (System) > 远程桌面 (Remote Desktop)。
    2. 将 远程桌面 的开关设置为 开 (On)，并在弹出的确认框中点击“确认”。
    3. 关键点： 在同一页面下方，记住这台电脑的 “设备名称”（例如：DESKTOP-12345）。
    4. 确保台式机设置了登录密码。远程桌面出于安全考虑，默认不支持无密码账户登录。

    第二步：检查网络和睡眠设置

    远程连接的前提是两台设备在同一个局域网（比如连着同一个 Wi-Fi 或路由器），且台式机不能处于休眠状态。

    1. 关闭休眠： 在台式机的 设置 > 系统 > 电源和电池 中，确保“在接通电源时，使设备进入睡眠状态”设置为 从不。
    2. 网络类型： 确保两台电脑的网络属性都设置为 “专用 (Private)”，而不是“公用 (Public)”，否则防火墙可能会拦截连接。

    局域网连接

    笔记本电脑

    1. 点击任务栏搜索框，输入 mstsc 或 “远程桌面连接” 并打开。
    2. 在“计算机”一栏，输入你刚才记下的台式机 设备名称（或者台式机的局域网 IP 地址）。
    3. 点击 连接。
    4. 系统会提示你输入凭据：
      • 用户名： 台式机的登录用户名（如果是微软账号，通常是邮箱地址）。
      • 密码： 台式机的登录密码。
    5. 如果弹出安全证书警告，直接点击“是”即可进入画面。

    非局域网连接

    第一步：建立虚拟的局域网

    使用异地组网工具 Tailscale

    给你的笔记本和台式机建立一个虚拟的局域网，让它们觉得彼此还在同一个房间里。

    • 操作步骤：
      1. 在两台电脑上都下载并安装 Tailscale。
      2. 用同一个账号（如 GitHub 或 Microsoft 账号）登录。
      3. 登录后，Tailscale 会给每台设备分配一个专用的 100.x.x.x 开头的 IP 地址。

    第二步：连接

    在笔记本的“远程桌面连接”里，直接输入台式机在 Tailscale 里的那个 IP 即可。

    Windows 11 Pro 原生远程桌面的“防窥”优势

    原生 RDP 与向日葵、TeamViewer 等第三方软件最大的不同在于：它在连接时会自动锁定物理屏幕。

    以下是几个层级的“隐藏”方案，你可以根据需求选择：

    1. 初级：原生 RDP 的自动锁定（默认状态）

    当你从笔记本成功连接到台式机的那一刻，台式机的物理显示器会立刻跳回 Windows 登录界面（锁定状态）。

    • 别人看到的是： 电脑锁屏了，显示的是你的壁纸和登录框。
    • 别人看不到的是： 你的鼠标在动、你打开了什么窗口、你在输入什么。
    • 提示： 只要你不把台式机的账号密码告诉旁边的人，他们绝对进不去，也看不见你的操作。
    1. 中级：直接关闭显示器电源

    这是最简单、最物理的方法。

    • 操作： 远程连接之前，直接伸手把台式机显示器的电源关掉。
    • 原理： RDP 远程桌面是“虚拟驱动”渲染画面的，它并不依赖物理显示器是否通电。即使显示器黑着，你的笔记本依然能看到流畅的画面。
    • 优点： 绝对安全，省电，且别人完全不知道电脑是开着的还是关着的。
